# Currency-Converter-
# 💱 Currency Converter - Конвертер валют

Приложение с графическим интерфейсом для конвертации валют в реальном времени с использованием актуальных курсов из API. Программа сохраняет историю конвертаций и позволяет работать с JSON-файлами.

## 👨‍💻 Автор

**Дидякина Кристина Игоревна**  
**Год разработки:** 2026

## 📖 Описание программы

Currency Converter - это desktop-приложение, которое позволяет:

- 💱 Конвертировать любые валюты мира по актуальным курсам
- 📊 Просматривать историю всех конвертаций
- 💾 Сохранять историю в JSON-файл и загружать её
- 🌐 Использовать более 160 валют (все поддерживаемые ExchangeRate API)
- 📈 Отслеживать курс конвертации для каждой операции
- 🎨 Работать в современном тёмном интерфейсе

## 🎯 Цель проекта

Создание удобного инструмента для быстрой конвертации валют с сохранением истории операций и возможностью экспорта данных.

📝 Примеры использования
Пример 1: Простая конвертация USD → EUR
Задача: Узнать, сколько будет 100 долларов в евро

Выберите "ИЗ ВАЛЮТЫ:" → USD

Выберите "В ВАЛЮТУ:" → EUR

Введите "СУММА:" → 100

Нажмите "КОНВЕРТИРОВАТЬ"

Результат: примерно 92.50 EUR

Пример 2: Конвертация RUB → KZT
Задача: Поездка в Казахстан, нужно перевести 5000 рублей в тенге

ИЗ ВАЛЮТЫ: RUB

В ВАЛЮТУ: KZT

СУММА: 5000

**КОНВЕРТИРОВАТЬ`

Результат: примерно 24250 KZT

Пример 3: Сохранение истории конвертаций
Задача: Сохранить все сегодняшние конвертации для отчёта

Выполните несколько конвертаций (все они автоматически сохранятся)

Нажмите "СОХРАНИТЬ В JSON"

Файл currency_history.json сохранится в папке с программой

При следующем запуске история загрузится автоматически

Пример 4: Загрузка ранее сохранённой истории
Задача: Загрузить историю конвертаций из другого файла

Нажмите "ЗАГРУЗИТЬ ИЗ JSON"

Выберите нужный JSON-файл

История загрузится и отобразится в таблице

Пример 5: Экзотическая конвертация
Задача: Перевести 50 британских фунтов в японские иены для планирования поездки

ИЗ ВАЛЮТЫ: GBP

В ВАЛЮТУ: JPY

СУММА: 50

**КОНВЕРТИРОВАТЬ`

Результат: примерно 9500 JPY

🛠️ Возможные ошибки и их решение
Ошибка	Причина	Решение
ModuleNotFoundError: No module named 'requests'	Не установлена библиотека requests	Выполните: pip install requests
"Неверный API ключ"	Проблема с API	Проверьте ключ в коде (переменная self.api_key)
"Нет подключения к интернету"	Отсутствует интернет	Подключитесь к интернету
"Введите корректное число"	В поле суммы буквы или символы	Введите только цифры и точку (например: 100.50)
"Превышено время ожидания"	Медленный интернет	Проверьте соединение, повторите попытку
"Файл не найден"	Нет сохранённого файла	Сначала сохраните историю
